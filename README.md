Chess Game Engine

Chess Game Engine is a Java-based mini project that demonstrates the fundamentals of Object-Oriented Programming (OOP) and Exception Handling. The application provides a simple menu-driven interface to start a game, display the chess board, make moves, and view the game status.

Features
Start a new chess game
Display the chess board
Make chess moves
View game status
Menu-driven interface
Exception handling for invalid inputs
Technologies Used
Java
OOP Concepts
Exception Handling
OOP Concepts Implemented
Class & Object
Abstraction
Encapsulation
Inheritance
Polymorphism
How to Run

Compile the program:

javac ChessGameEngine.java

Run the program:

java ChessGameEngine
Output

The program displays a menu that allows users to perform chess operations and handles invalid inputs gracefully using exception handling.
